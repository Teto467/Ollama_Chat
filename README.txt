Ollama Chat セットアップガイド
セットアップ手順

Ollamaのインストール手順
Ollamaのダウンロード
OllmaDownload を開き、Ollamaをダウンロードしてインストールしてください。

モデルのインストール手順
モデルリストの確認
OllamaModelsList を開き、使用したいモデルを選択してください。

モデルのダウンロードと実行
新しいコマンドプロンプトを開き、以下の様なコマンドを実行してモデルをダウンロードします。
（例: ollama run deepseek-r1）
ダウンロードが完了するまで待機してください。

インストール完了の確認
SendMassage と表示されたら、AIモデルのインストールが完了しています。

ライブラリのインストール
install_libraries.bat を実行してください。


OllamaChatの実行
run_ollama_sequence.bat を実行してください。
